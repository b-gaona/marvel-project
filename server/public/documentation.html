Marvel Snap API

Documentation

Introduction

Welcome to the Marvel Snap API! This documentation should help you familiarise yourself with the resources available and how to consume them with HTTP requests. The data that this API has is from the dataset found here in Kaggle . Read through the getting started section before you dive in. Most of your problems should be solved just by reading through it.

Getting started

Let's make our first API request to the Marvel Snap API! Open up a terminal and use curl or Postman to make an API request for a resource. In the example below, we're trying to get the first card, Abomination:

http://marvel-snap-bgaona.onrender.com/v1/cards/1/

We'll use Postman for our examples as it displays responses nicely and gives us a whole lot more useful information. If you don't want to download Postman, just use the curl command instead.

Here is the response we get:

Content-Type: application/json
{
  "_id": "6394b612bdd8b00592b40efc",
  "cid": 1,
  "__v": 0,
  "art": "https://marvelsnapzone.com/wp-content/themes/blocksy-child/assets/media/cards/abomination.webp?v=21",
  "cname": "Abomination",
  "cost": 5,
  "power": 9,
  "ability": ""
}

If your response looks slightly different don't panic. This is probably because more data has been added to Marvel Snap API since we made this documentation.

Base URL

The Base URL is the root URL for all of the API, if you ever make a request to Marvel Snap API and you get back a 404 NOT FOUND response then check the Base URL first.

The Base URL for Marvel Snap API is:

http://marvel-snap-bgaona.onrender.com/v1/

The documentation below assumes you are prepending the Base URL to the endpoints in order to make requests.

JSON Schema

All resources support JSON Schema. Making a request to /v1/<resource>/schema will give you the details of that resource. This will allow you to programmatically inspect the attributes of that resource and their types.

Encodings

JSON

JSON is the standard data format provided by Marvel Snap API by default.

Resources

Root

The Root resource provides information on all available resources within the API.

Example request:

http://marvel-snap-bgaona.onrender.com/v1/

Example response:

Content-Type: application/json
{
    "cards": "http://marvel-snap-bgaona.onrender.com/v1/cards/",
    "decks": "http://marvel-snap-bgaona.onrender.com/v1/decks/",
}

Attributes:

  • cards string -- The URL root for Card resources.
  • decks string -- The URL root for Deck resources.

Cards

A Card resource is an individual card within the Marvel Snap.

Endpoints

  • /cards/ -- Get all the card resources.
  • /cards/:id/ -- Get a specific card resource.
  • /cards/schema/ -- View the JSON schema for this resource.

Example request:

http://marvel-snap-bgaona.onrender.com/v1/cards/10

Example response:

Content-Type: application/json
{
    "_id": "6394b611bdd8b00592b40b9b",
    "cid": 10,
    "__v": 0,
    "art": "https://marvelsnapzone.com/wp-content/themes/blocksy-child/assets/media/cards/angela.webp?v=21",
    "cname": "Angela",
    "cost": 2,
    "power": 1,
    "ability": "When you play a card here, +2 Power."
}

Attributes:

  • _id string -- The object id in the database of this card.
  • cid number -- The id of this card.
  • __v number -- The corresponding version of this card.
  • art string -- The image path of this card.
  • cname string -- The name of this card.
  • cost number -- The cost of this card.
  • power number -- The power of this card.
  • ability string -- The ability of this card.

Decks

A Deck resource is an individual deck within the Marvel Snap.

Endpoints

  • /decks/ -- Get all the deck resources.
  • /decks/:id/ -- Get a specific deck resource.
  • /decks/schema/ -- View the JSON schema for this resource.

Example request:

http://marvel-snap-bgaona.onrender.com/v1/decks/121

Example response:

Content-Type: application/json
{
  "_id": "63979efdfde8e6810d7f460c",
  "name": "Buffs",
  "__v": 0,
  "avg_cost": 3.3,
  "avg_power": 3,
  "cids": [
      {
          "_id": "6394b610bdd8b00592b40852",
          "cid": 181,
          "__v": 0,
          "art": "https://marvelsnapzone.com/wp-content/themes/blocksy-child/assets/media/cards/nightcrawler.webp?v=21",
          "cname": "Nightcrawler",
          "cost": 1,
          "power": 2,
          "ability": "You can move this once."
      },
      ...
  ],
  "did": 121,
}

Attributes:

  • _id string -- The object id in the database of this deck.
  • did number -- The id of this deck.
  • __v number -- The corresponding version of this deck.
  • cids array -- The corresponding cards of this deck.
  • name string -- The name of this deck.
  • avg_cost number -- The average cost of this deck.
  • avg_power number -- The average power of this deck.
